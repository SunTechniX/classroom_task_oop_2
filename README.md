# Задания по ООП в Python

## Задание 1: Система уведомлений

Создайте абстрактный класс `Notifier` с методом `send(message: str)`.  
Реализуйте три подкласса:
- `EmailNotifier`
- `SMSNotifier`
- `PushNotifier`

Каждый должен выводить сообщение в формате:  
`[Email] Отправка: {message}`  
`[SMS] Отправка: {message}`  
`[Push] Отправка: {message}`

Напишите функцию `notify_all(notifiers, message)`, которая отправляет сообщение через все переданные способы.

> Вывод должен выполняться с помощью `print()`.

## Задание 2: Животные и наследование

Создайте базовый класс `Animal`:
- Конструктор `__init__(self, size)` сохраняет `size` в `self.size`.
- Метод `speak(self)` — пустой (возвращает `None`).
- Метод `about_me(self)` возвращает строку `'Я животное!'`.

Создайте два подкласса:
- `Dog` — переопределяет `speak()` → `'Гав!'`
- `Cat` — переопределяет `speak()` → `'Мяу!'`

**Важно**: в `Dog` и `Cat` **не должно быть** методов `__init__` и `about_me`.

Создайте объекты `Dog(size1)` и `Cat(size2)`.  
Выведите для каждого одну строку в формате:
```
Гав! Я животное! Мой размер: 7
Мяу! Я животное! Мой размер: 3
```

Составьте строку в `print()` так:
```python
print(dog.speak(), dog.about_me(), f"Мой размер: {dog.size}")
```
